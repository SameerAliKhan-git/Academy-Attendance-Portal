About Academy Attendance System

Project Overview

The Academy Attendance System is a comprehensive web-based solution designed to streamline attendance tracking and communication between students, parents, and faculty members.

Key Features
  • Real-time attendance tracking and monitoring
  • Role-based dashboards for different user types
  • Department announcements and updates
  • Assignment posting with file attachments
  • Important notices and alerts
  • Detailed attendance logs and reports
  • Mobile-responsive design
User Roles
Students
  • View attendance percentage
  • Access detailed attendance logs
  • View announcements and assignments
  • Download shared files
Parents
  • Monitor child's attendance
  • View all school communications
  • Access assignments and notices
  • Download shared documents
Teachers
  • Manage department students
  • Mark and track attendance
  • Create announcements and assignments
  • Upload files for students
Administrators
  • Full system access
  • User management
  • Department administration
  • Generate system reports
Technology Stack
Backend
  • Flask 3.0.0 (Python Web Framework)
  • SQLAlchemy (Database ORM)
  • Flask-Login (Authentication)
  • Flask-WTF (Forms)
Frontend
  • HTML5 & CSS3
  • Bootstrap 5 (UI Framework)
  • JavaScript (Interactivity)
  • Bootstrap Icons
Development Team

Team B - Web & Dashboard Development

Responsible for designing and developing the user-facing dashboards, implementing authentication systems, integrating with backend APIs, and ensuring a responsive, user-friendly experience across all devices.
